# RCJA Rescue Maze — Mighty Maisy 2026

This is not a RCJA affiliated project. Built with the help of Claude and follows the RCJA Maze Rescue Rules 2026.

## Simulator

Open [RCJA Maze Simulator](https://winglander.github.io/RCJA-Maze-Simulator/sandbox.html) to launch the simulator.

**Features:**
- **Click-to-cycle maze design** — click a tile centre to cycle Empty ⬜ → Green 🟩 → Red 🟥 → Black ⬛; click a tile edge to toggle walls
- **Blockly visual block editor** — drag-and-drop blocks for programming robot behaviour (sensors, motors, timing, actions)
- **JavaScript code editor** — write code directly with full API access
- **Simulated sensors** — three VL53LIX-style ToF distance sensors and a TCS34725-style colour sensor
- **Live scoring** — tracks victims found, indications, exit bonus, and count bonus per RCJA rules
- **Maze protection** — toolbar locks during a run to prevent accidental edits
- **Import/Export** — save and load mazes as JSON

Please be aware that a simulated environment is very different to a live environment. Use this as a proof-of-concept and leave plenty of time to build a physical robot for competition.

## Scorer

Open [RCJA Maze Scorer](https://winglander.github.io/RCJA-Maze-Simulator/scorer.html) for Mighty Maisy Maze and Intermediate Maze scoring.

This scorer follows the RCJA Maze Rescue Rules 2026, created for the convenience of teachers who wish to keep score of their own competitions. This is NOT an official RCJA scoring system.
