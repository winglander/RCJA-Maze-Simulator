# RCJA-Maze-Simulator
This is not a RCJA affiliated project.

Open index.html to start the RCJA Maze Simulator. This was written with the help of Claude and follows the RCJA Maze Rescue Rule 2026.

Please be aware that a simulated environment is very different to live environment. You should only use this as proof-of-concept and leave plenty of time to build a physical robot for competition.

# RCJA-Maze-Scorer
Open scorer.html to start RCJA-Maze-Scorer for Mighty Maisy Maze and Intermediate Maze Scoring.

This scorer follows the RCJA Maze Rescue Rule 2026, and created for the convenience of teachers who wish to keep score of their own little competition. This is NOT an official RCJA scoring system.
